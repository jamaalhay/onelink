What is verified

COD checkout works, order success/tracking pages load, SMS fallback delivered with correct totals, and WhatsApp links open to the real Onelink number.

What to skip unless asked

Card payment is live Stripe and charges real cards. Do not run card tests unless Jamaal explicitly approves a real charge/refund test.

Support number

WhatsApp CTAs should point to 18763960575. If you see any other number, flag it.

Before You Start

Use Chrome desktop first, then one mobile browser pass. Keep screenshots for anything odd.

Core Storefront Smoke

Confirm the main customer experience is stable before placing orders.

Checkout Validation

COD is the primary live test. Card is real-money and optional only with explicit approval.

Do not use Stripe test card numbers on live Stripe. If Jamaal asks you to test card payment, use a real card on a low-value cart, capture the exact error/success state, and confirm refund/cancel handling in admin.
Known card behavior: card authentication failures may appear as "We are unable to authenticate your payment method." Treat that as known unless the page crashes, charges unexpectedly, or creates an order incorrectly.

Notification Validation

Validate what a customer actually receives after checkout.

Post Purchase

Verify the customer can recover and follow their order after checkout.

Admin Validation

Use admin credentials from Jamaal. Keep admin open in a separate tab while testing checkout.

Mobile And Performance Pass

One phone pass is enough unless you find layout trouble.

Known Issues And Non-Blockers

Do not file these unless behavior is worse than described.

Issue Report Template

Copy this for each issue. One issue per report is best.

Flag immediately: any 500 error, order succeeds but does not appear in admin, SMS/email never arrives after 2 minutes, wrong order total in message/page/admin, real charge without a matching admin order, or Add to Cart failing for an in-stock product.
This checklist stores checkbox state locally in the browser only. It does not send results anywhere.
